# Autoclass
这是一格自动化学习课程。
version规则
    Vx.y.z
    x对应于大版本
    y对应基于大版本的完善工作Autoclass++
    z为面向对象（z == 2为公开；z == 1为私人；其余为定做）
version content
    V1.2.2
        1.在终端输入网址账号密码，注意：网址是科目（如大数，大学物理）下课程（微积分的基本应用，速度的向量）页面下的网址！！！
        2.此版本需要自行手动编辑test.json文件，将需要看的课程名（title标签，建议利用开发者工具点击复制）完整的粘贴进去，要求：每行填入一个且顶格写入。
        3.data.exe/py 文件用于编辑test.json文件。
        4.只以edge浏览器为驱动，作为唯一基准。
    V1.3.2
        add：
            1.Google浏览器驱动
            2.弹幕答题优化完善。
    V1.4.2
        add：
            1.data.exe/py 文件用于编辑text.json文件。并且现在为自动编辑写入。但还请多加留意在json文件中删除不必要课程防止报错。
